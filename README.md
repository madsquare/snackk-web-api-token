# snackk-web-api-token

## require
* jquery-cookie

## not amd
```
window.snackkModule.token
```

## functions
token:
* **getAccessToken** : get access-token from cookie
* **getRefreshToken** : get refresh-token from cookie
* **setToken** : set access-token and refresh-token to cookie

sid:
* **generateSid**
* **getSid** : get sid
